# data-structures-2021

## MyArrayList
A self resizing array data structure implementing:
### public methods
- isEmpty()
- at(index)
- push(value)
- insert(index, value)
- prepend(value)
- pop()
- delete(index) 
- remove(value) - all occurences
- find(value)
- toString()
### private methods
- resize(newCapacity)
